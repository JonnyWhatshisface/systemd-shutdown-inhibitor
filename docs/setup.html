Getting Started

Install, enable and verify terminusd

terminusd is available either by source or packages for Fedora, Red Hat and Debian.

Package Repositories

Debian and RPM packages are available on the Github in the releases.

Repositories are on the way.

Install from source/h2> 
make
sudo make install
sudo terminusctl start
[configure] Configure /etc/terminusd.conf and /etc/terminus.d/*.conf drop-ins as needed
[optional] terminusctl --set-logind-inhibitor-delay

Installed file layout

  • /usr/sbin/terminusd
  • /usr/sbin/terminusctl
  • /usr/lib/systemd/system/terminusd.service
  • /etc/terminusd.conf
  • /etc/terminus.d drop-in directory
  • /usr/libexec/terminusd/examples example scripts
  • /usr/share/man/man8/terminusd.8
  • /usr/share/man/man8/terminusctl.8

Initial validation

sudo terminusctl status
Terminusd status should show the current running configuration.
Residual configurations may be present if previously installed.
